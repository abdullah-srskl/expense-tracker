# Expense Tracker

Terminal tabanlı kişisel harcama takip uygulaması. Python ile geliştirilmiştir ve harcamaları JSON dosyasında saklar.

## Özellikler

- Harcama ekleme
- Harcamaları listeleme
- Toplam harcama görüntüleme
- Kategoriye göre filtreleme
- Harcama silme
- Harcama güncelleme
- Kategoriye göre çubuk grafik gösterme
- Kategoriye göre pasta grafik gösterme
- JSON ile veri kaydetme ve yükleme
